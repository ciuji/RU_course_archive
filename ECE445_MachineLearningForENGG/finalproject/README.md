# ECE445_MLForEngg_Project

- a repo of the ECE445 final project.

- created by Chaoji Zuo, Changlin Jiang, Shuyu Chen.

  - Classification problem. Using the dataset of [blood-cells](https://github.com/Shenggan/BCCD_Dataset)
